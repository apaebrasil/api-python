API CEBAS

  • [GET] /api/cebas?cnpj=

    -> Retorna os Dados Buscando pelo CNPJ - Parâmetro (cnpj) ex: 62388566000190

  • [GET] api/cebas/getcount

    -> Retorna a quantidade de linhas totais

  • [GET] /api/cebas/all

    -> Retorna todos os dados do cebas

  • [GET] /api/cebas/getlink

    -> Retorna um link em formato Json para baixar a planilha do cebas atualizado

  • [GET] /api/cebas/paginado?row_start=1&qtd_rows=5&status=DEFERIDO&dt_inicio=2024-05-02&dt_fim=2024-12-31

    -> Retorna os Dados Paginados - Parâmetros (row_start) (qtd_rows)
